ExpressJS is a nodejs web applicaation framework that provides a cleaner and simple way to write nodejs. it also provides tools and features to build fast web servers and API's easily
why ExpressJS?
  ExpressJS is the standard framework for nodejs because
  i. simplicity: ExpressJS provides simple functions for routing and middleware instead of manually using nodejs http module
  ii. Fast:it is built directly based on nodejs core modules so its fast
  iii. middleware: using express , we can add extra middle functionalities like authentication , logging etc.., easily
  iv. express is unopiniated means we can work as any folder structure and with any database
  v.community: it contains tons of modules and lot of community.

  How ExpressJS Works?

   When client requests, express passes the request through some functionalities(middleware), then it will go to particular route and based on logic, it will send response. if it dowsn't send any response, then automatically it calls next() to pass the request to next middleware

create basic API server using express js
import express from 'express'
const app = express(); // it creates an express appplication which is used to create API's
app.use(express.json())  // middleware for parsing
app.get('/', (req,res) =>
{
	res.send({"message":"Backend setup successfully"})
})
const port = process.env.PORT ||5173

let products = [

app.get('/products', (req, res) =>
{
	res.status(200).send(products)
})

app.post('/products', (req, res) =>
{
	console.log(req.body)
	const { name, price, category, inStock } = req.body;
	if (!name || !price || !category || !inStock)
	{
		return res.status(401).json({message:"Please enter name, price, category and instock. something is missing"})
	}
	const newProduct = {
		id: "PROD00" + products.length + 1,
		name: name,
		price: price,
		category: category,
		inStock:inStock
	}
	products.push(newProduct)
	res.status(201).json({message:"Successfullly added",newProduct})
})

// for updating we do put request
app.put('/products/:id', (req, res) =>
{
	const idValue = req.params.id
	console.log(idValue)
	const ele = products.find((e) => e.id == idValue)
	console.log(ele)
	const {name,price,category,isStock} = req.body
	if (name) { ele.name = name }
	if (price) { ele.price = price }
	if (category) { ele.category = category }
	if(isStock){ele.isStock = isStock}
	res.send({message:"ok",ele})
})
// for deleteing we do delete request
app.delete('/products/:id', (req, res) =>
{
	const idValue = req.params.id
	console.log(idValue)
	const ele = products.findIndex((e) => e.id == idValue)
	console.log(ele)
	products.splice(ele,1)
	res.send({message:"Succesfuly deleted"})
})
app.listen(5173, () =>
{
	console.log("it is running at http://localhost:5173")
})
